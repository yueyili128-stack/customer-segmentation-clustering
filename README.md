
# Customer Segmentation with Clustering

This project implements an end-to-end customer segmentation pipeline using unsupervised machine learning techniques.
The goal is to group customers based on behavioral and demographic features to support targeted marketing strategies.

## Features
- Data preprocessing and feature scaling
- K-Means and Hierarchical Clustering
- Elbow Method & Silhouette Score for evaluation
- Cluster visualization and interpretation
- Modular, production-oriented project structure

## Tech Stack
- Python
- pandas, numpy
- scikit-learn
- matplotlib, seaborn

## How to Run
```bash
pip install -r requirements.txt
python main.py
